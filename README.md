# 饭友地图局

一个完全静态、所有匹配与计算都在浏览器本地完成的“高中毕业蹭饭地图”。

## 使用

直接打开 `index.html` 即可。为了获得完整的剪贴板与下载体验，推荐在目录中启动一个静态服务器：

```bash
python3 -m http.server 4173
```

然后访问 <http://localhost:4173>。

## 特性

- 内置常见高校和 300+ 国内城市的坐标，无需地理编码 API
- 支持手动经纬度，覆盖内置词典未收录的地点
- 支持自定义班级地图标题与海报标语，并保存在当前浏览器
- 地图用“小圆点 + 姓名全文”标出每位同学，同地点采用有最大半径限制的紧凑散开
- 预览区与导出海报统一采用中央地图、左右地区名单的结构，并可直接查看或删除单人
- 导出 PNG 或可无限放大的 SVG，均采用“中央地图 + 四周引导标注”的环绕版式
- 提供米白简报、青春蓝图、深色海报和黑白打印四种导出样式
- 内置 50 人完整姓名示例，适合班级规模展示
- 本地打包完整省级行政区矢量边界，含台湾省、港澳和南海诸岛附图
- 数据仅存放在当前浏览器 `localStorage`
- 一键复制 JSON 数据、导出 PNG 或 SVG
- 无框架、无 CDN、无构建步骤，可部署到任意静态托管服务

## 地图合规说明

行政区矢量数据已打包在 `china-map.js`，页面运行时不发起网络请求。公开发布含新增标注的地图前，请根据《公开地图内容表示规范》和地图审核相关规定履行审核程序；请勿为未经审核的二次制图自行标注审图号。

城市中心坐标整理自 MIT 许可的 [ChinaGeoJson / DataV.GeoAtlas](https://github.com/zhChuXiao/ChinaGeoJson)，已静态打包在 `city-data.js` 中。
